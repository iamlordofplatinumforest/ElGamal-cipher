# Криптосистема Эль-Гамаля

Реализация криптосистемы Эль-Гамаля с графическим интерфейсом на Python.

## Описание

Криптосистема Эль-Гамаля - асимметричный алгоритм шифрования, основанный на сложности вычисления дискретного логарифма.

## Возможности

- Проверка чисел на простоту (тест Миллера-Рабина)
- Поиск первообразных корней
- Генерация ключевых пар
- Шифрование и дешифрование файлов
- Поддержка форматов: TXT, DOCX, PNG, JPEG

## Установка

```bash
pip install -r requirements.txt
```

## Использование

```bash
# Новый модульный запуск
python 3lab/main.py

# Или старый способ (обратная совместимость)
python 3lab/ElGamalApp.py
```

## Структура

### Основные модули
- `main.py` - главное приложение
- `prime_checker.py` - проверка чисел на простоту
- `number_theory.py` - математические операции теории чисел
- `key_generator.py` - генерация ключевых пар
- `cipher.py` - шифрование и дешифрование
- `file_manager.py` - работа с файлами
- `crypto_parameters.py` - управление параметрами

### GUI модули
- `parameters_tab.py` - вкладка параметров
- `encryption_tab.py` - вкладка шифрования
- `decryption_tab.py` - вкладка дешифрования

### Совместимость
- `Algorithm.py` - обратная совместимость со старым кодом
- `ElGamalApp.py` - старый монолитный файл (сохранен)
- `requirements.txt` - зависимости

## Алгоритм

1. Выбирается простое число p
2. Выбирается первообразный корень g
3. Генерируется закрытый ключ x
4. Вычисляется открытый ключ y = g^x mod p
5. Шифрование: a = g^k mod p, b = (y^k * m) mod p
6. Дешифрование: m = (b * a^(-x)) mod p
